osc_unity_python
================

test for osc between unity and python on windows


pyosc/OSC.py is the module for python to use OSC. Please set the right path to import it. 
pyosc/unity-python-osc.py is the sample of OSC communication.

OSC_TEST_FOR_UNITY is the project of unity which use OSC. 
- right click to send  calibration start flag to python through OSC
- left click to send calibration step message to python
- ÅgSÅh is a test command to start message sending from python to unity.
- ÅgqÅh is a test command to stop python process.
